    Le tue foto:
    perché dovrebbero finire su un server altrui?

    I servizi cloud catalogano le tue immagini a distanza. OffGallery fa tutto in locale.

    Cloud = perdita di controllo

    I servizi cloud di catalogazione AI caricano le tue immagini su server remoti. I tuoi soggetti, le tue location, i tuoi clienti: metadati estratti lontano dai tuoi occhi.

    Abbonamenti e lock-in

    Paghi ogni mese per funzionalità che potrebbero sparire. Il giorno in cui smetti di pagare, perdi l'accesso ai tuoi stessi dati catalogati.

    Ricerche farraginose

    La maggior parte dei software cerca per nome file o tag manuali. OffGallery capisce il contenuto visivo: "tramonto con riflesso sull'acqua" trova le foto giuste.

    OffGallery: l'alternativa.

    Modelli AI di nuova generazione girano in locale su hardware consumer. OffGallery li orchestra tutti — CLIP, DINOv2, BioCLIP, LLM vision — senza connessione internet.

    Progettato per chi lavora seriamente con le immagini.

    Cinque funzionalità core, tutte eseguite in locale sul tuo hardware.

    Privacy

    100% offline e privato

    Nessuna foto caricata online, mai. Tutti i modelli AI girano in locale. Dopo il primo download, OffGallery funziona senza connessione internet.*

    * Eccetto l'installazione iniziale dei modelli AI (~6 GB, una tantum) e, se richiesto, alcuni plugin.

    AI

    AI multi-modello

    CLIP per la ricerca semantica, DINOv2 per la similarità visiva, BioCLIP per ~450.000 specie biologiche, LLM vision locale per titoli e tag automatici, Aesthetic scorer e MUSIQ per la qualità tecnica ed estetica.

    Integrazione

    Integrazione Lightroom

    Importa cataloghi .lrcat e sincronizza tag, rating e metadati via sidecar XMP compatibili con Lightroom, Capture One e qualsiasi DAM che supporti lo standard.

    Ricerca

    Ricerca semantica

    Cerca per concetto: "bambino che corre sulla spiaggia", "volatili in volo al tramonto". La query viene compresa e confrontata con il contenuto visivo di ogni immagine.

    Plugin

    Plugin ecosystem

    Architettura estendibile tramite plugin. Scegli il backend LLM tra Ollama e LM Studio, e arricchisci le funzionalità con moduli specializzati come BioNomen per la fotografia naturalistica.

    Backend LLM

    Ollama LLM Vision

    Collega OffGallery a Ollama per generare tag, titoli e descrizioni. Supporta modelli vision come Qwen3-VL, LLaVA, Gemma3. Accesso riservato ai beta tester.

    Backend LLM

    LM Studio

    Usa LM Studio come server LLM locale con interfaccia grafica. Ideale per gestire modelli vision quantizzati. Accesso riservato ai beta tester.

    Fotografia naturalistica

    BioNomen

    Arricchisce i tag con i nomi comuni delle specie biologiche in più lingue, attingendo al database GBIF. Indispensabile per wildlife e fotografia botanica.
