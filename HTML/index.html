Le tue foto: perché dovrebbero finire su un server altrui?

  • I servizi cloud catalogano le tue immagini a distanza. OffGallery fa tutto in locale.
  • Lascia il compito della catalogazione al tuo computer.
  • Con gli appositi plugin, personalizza il tuo sistema e aggiungi nuove funzionalità.

Cloud = perdita di controllo

I servizi cloud di catalogazione AI caricano le tue immagini su server remoti. I tuoi soggetti, le tue location, i tuoi clienti: metadati estratti lontano dai tuoi occhi.

Abbonamenti e lock-in

Paghi ogni mese per funzionalità che potrebbero sparire. Il giorno in cui smetti di pagare, perdi l'accesso ai tuoi stessi dati catalogati.

Cerchi per parole chiave, trovi frustrazione.

La maggior parte dei software cerca per nome file o tag manuali. OffGallery capisce il contenuto visivo: "tramonto con riflesso sull'acqua" trova le foto giuste, anche senza keywords (tags).

L'alternativa esiste.

Modelli AI di nuova generazione girano in locale su hardware consumer. OffGallery li orchestra tutti — CLIP, DINOv2, BioCLIP, LLM vision — senza connessione internet.

NUOVO
Installazione

One-click su Windows e Linux

OffGallerySetup.exe (Windows) e OffGallerySetup (Linux) installano tutto in automatico: Miniconda, Python, librerie e modelli AI (~8 GB). Nessun terminale. Nessuna configurazione.

macOS: script guidato nella cartella installer/.

Privacy

100% offline e privato

Nessuna foto caricata online, mai. Tutti i modelli AI girano in locale. Dopo il primo download, OffGallery funziona senza connessione internet.*

* Eccetto l'installazione iniziale dei modelli AI (~6 GB, una tantum) e, se richiesto, alcuni plugin.

AI

AI multi-modello

CLIP per la ricerca semantica, DINOv2 per la similarità visiva, BioCLIP per ~450.000 specie biologiche, LLM vision locale per titoli e tag automatici, Aesthetic scorer e MUSIQ per la qualità tecnica ed estetica.

Integrazione

Integrazione Lightroom

Importa cataloghi .lrcat e sincronizza tag, rating e metadati via sidecar XMP compatibili con Lightroom, Capture One e qualsiasi DAM che supporti lo standard.

Ricerca

Ricerca semantica

Cerca per concetto: "bambino che corre sulla spiaggia", "volatili in volo al tramonto". La query viene compresa e confrontata con il contenuto visivo di ogni immagine.

Plugin

Plugin ecosystem

Architettura estendibile tramite plugin. Scegli il backend LLM tra Ollama e LM Studio, e arricchisci le funzionalità con moduli specializzati come BioNomen per la fotografia naturalistica.

★ Feature esclusive disponibili tramite plugin

Alcune delle capacità più uniche di OffGallery sono disponibili attraverso il sistema plugin. Tra queste, il generatore AI di prompt specializzati per la generazione di tag, titoli e descrizioni, adattati a specifiche esigenze: basterà descrivere in linguaggio naturale le proprietà richieste e l'AI svilupperà un prompt ottimizzato, sostitutivo di quello standard, col quale guiderà l'LLM Vision nella produzione di tag, titoli e descrizioni delle immagini sottoposte — una funzione che non esiste in nessun altro software di catalogazione fotografica.

L'interfaccia.

Estendi OffGallery con moduli specializzati.

Ogni plugin viene riconosciuto automaticamente all'avvio. Scegli il backend LLM, attiva solo quello che usi — gli altri non occupano VRAM.

Backend LLM

Ollama LLM Vision

Collega OffGallery a Ollama per generare tag, titoli e descrizioni. Supporta modelli vision come Qwen3-VL, LLaVA, Gemma3.

Scarica (per Mac)

Backend LLM

LM Studio

Usa LM Studio come server LLM locale con interfaccia grafica. Ideale per gestire modelli vision quantizzati.

Plugin di Riccardo Merlotti

Scarica (per Mac)

Fotografia naturalistica

BioNomen

Un airone cenerino identificato da BioCLIP diventa automaticamente "grey heron", "héron cendré", "Graureiher" — in tutti i sidecar XMP, offline. Dati GBIF bundled, taxa selezionabili dalla configurazione.

Scarica (per Mac)

Fotografia naturalistica

GeoSpecies

Affina la classificazione BioCLIP usando solo le specie attese nella posizione GPS della foto. Restringe il campo da ~450.000 specie globali a un sottoinsieme geografico pertinente, aumentando la precisione tassonomica. Checklist scaricate da GBIF, poi usate offline.

Scarica (per Mac)

Dati geografici

GeoNames

Assegna coordinate GPS e gerarchia geografica precisa (Continente › Paese › Regione › Città) a milioni di luoghi tramite il database GeoNames. Sostituisce il motore builtin con precisione superiore su borghi e frazioni.

Scarica (per Mac)

Dati geografici

NaturArea

Identifica l'area protetta (WDPA/UNEP) e il tipo di habitat (ESA WorldCover) per ogni foto con GPS. Filtra per habitat o area protetta direttamente dalla ricerca.

Scarica (per Mac)

Dati geografici

Weather Context

Recupera le condizioni meteo storiche allo scatto via Open-Meteo (temperatura, vento, pioggia, neve…). Le condizioni sono filtrabili dalla ricerca e visibili nelle imagecard della gallery.

Scarica (per Mac)

Backend LLM

Contesto Prompt

Inietta un blocco CONTEXT nel prompt Vision-Language per orientare il modello verso il dominio fotografico dell'archivio. Preset built-in (naturalistica, paesaggio, astro, macro, subacqueo, reportage…) e preset personalizzati generati dal modello locale.

Scarica (per Mac)

Tutti i plugin sono gratuiti.

Windows e Linux: installa i plugin direttamente dall'installer grafico di OffGallery, nella sezione «Plugin».
macOS: scarica lo zip del plugin dal pulsante «Scarica» sulla sua scheda qui sopra, poi estrai la cartella dentro plugins/ nella tua installazione di OffGallery. Al primo avvio ogni plugin scarica i propri dati (mappe, database) dalla sua configurazione.
